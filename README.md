# Flask Weather Example

Simple Flask app that lets a user enter a city and displays current weather using OpenWeatherMap.

Setup

1. Create and activate a Python environment (optional but recommended).

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Export your OpenWeatherMap API key:

```bash
export OPENWEATHER_API_KEY="your_api_key_here"
```

4. Run the app:

```bash
python app.py
```

Open http://127.0.0.1:5000 in your browser.
